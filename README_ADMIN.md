# دليل لوحة التحكم — سعودي كارييرز

## 1. قراءة نتائج AI Match 🤖

عند فتح تبويب **التقديمات**، يظهر شريط ملوّن أعلى كل بطاقة متقدم:

| اللون | النسبة | التفسير | الإجراء المقترح |
|---|---|---|---|
| 🟢 **أخضر** | ≥ 80% | مطابقة عالية — High Quality Lead | راجعه أولاً، حدّد مقابلة |
| 🟡 **أصفر** | 50–79% | مطابقة متوسطة | راجع رسالة التقديم يدوياً |
| 🔴 **أحمر** | < 50% | مطابقة منخفضة | يمكن تأجيله أو رفضه |

> **ملاحظة:** الدرجة تظهر فقط للمتقدمين الذين وافقوا على خانة "أوافق على التحليل بالذكاء الاصطناعي" عند التقديم.

## 2. الأبعاد السبعة للمطابقة

الدرجة الإجمالية مجموع مرجّح من 7 أبعاد:

| البُعد | الوزن الافتراضي | ما يقيسه |
|---|---|---|
| المهارات التقنية | 35% | تطابق كلمات الوظيفة مع رسالة التقديم |
| الخبرة | 25% | إشارات المستوى (مبتدئ/متوسط/كبير/قيادي) |
| الموقع | 15% | ذكر المدينة في رسالة التقديم |
| نوع الدوام | 10% | (عن بعد/كامل/جزئي) |
| التعليم | 7% | ذكر شهادة أو تخصص |
| القطاع | 5% | ذكر مجال الوظيفة |
| اكتمال الطلب | 3% | رفع CV + هاتف + رسالة |

## 3. تعديل الأوزان من لوحة التحكم

1. افتح **لوحة التحكم → تبويب ⚙️ الإعدادات**
2. ابحث عن مجموعة **AI Matching Weights**
3. عدّل قيمة أي وزن (رقم عشري بين 0 و 1)
4. اضغط 💾 بجانب كل حقل لحفظه

> **قاعدة:** مجموع الأوزان السبعة يجب أن يساوي 1.0  
> إذا لم تساوِ 1.0، يُعيد النظام توزيعها تلقائياً (normalization).

**مثال:** لتعطي وزناً أكبر للموقع في حملة توظيف محلية:
```
ai.weight_location  → 0.25  (رُفع من 0.15)
ai.weight_education → 0.05  (خُفض من 0.07)
ai.weight_category  → 0.03  (خُفض من 0.05)
```

## 4. حد الـ High Quality Lead

**الإعداد:** `ai.hql_threshold` (افتراضي: 80)

عند وصول متقدم لهذه الدرجة أو أعلى:
- يظهر الشريط الأخضر في لوحة التحكم
- يُطلق حدث `hql_application` في Google Tag Manager
- تستخدمه إعلانات Google وMeta لبناء جمهور مشابه (Lookalike)

لتشديد المعيار: غيّر القيمة لـ 85 أو 90  
لتخفيفه في بداية الإطلاق: غيّره لـ 70

## 6. حقن Google Analytics / GTM / Facebook Pixel

1. افتح **لوحة التحكم → ⚙️ الإعدادات → 📊 التحليلات والإعلانات**
2. أدخل المعرّفات:
   - `analytics.gtm_id` — بصيغة `GTM-XXXXXXX`
   - `analytics.ga_id` — بصيغة `G-XXXXXXXXXX` (إذا لم يكن GTM يديره)
   - `analytics.fb_pixel` — رقم الـ Pixel (15-16 رقماً)
3. اضغط 💾 — يُحقَن تلقائياً في الموقع فوراً دون إعادة نشر

---

## 7. تتبع الزوار (PDPL)

كل متقدم وافق على التحليل بالذكاء الاصطناعي يُظهر شارة **PDPL ✓** خضراء.  
الضغط على بطاقته يفتح **لوحة التفاصيل** التي تتضمن:
- تاريخ ورقم نسخة الموافقة القانونية
- زر **"سحب البيانات"** لتنفيذ طلب الحذف وفق نظام PDPL
